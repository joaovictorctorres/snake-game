# 🐍 Snake Game

![Preview do Jogo](./assets/preview.png)

> Jogo da cobrinha clássico reinventado com fases progressivas, obstáculos, fruta dourada com escudo e controle de zoom — tudo rodando no navegador com HTML, CSS e JavaScript puro.

🔗 **[Jogar agora](https://snake-game-nu-pied.vercel.app)**

---

## 🎮 Como jogar

- Use as **setas do teclado** para mover a cobra
- Pressione **Espaço** para iniciar, passar de fase ou reiniciar
- Coma as frutas vermelhas para pontuar e crescer
- Fique de olho na **fruta dourada** — ela concede um escudo!
- Chegue a **50 pontos** para avançar de fase

---

## ✨ Funcionalidades

- 3 fases com velocidade e obstáculos crescentes
- Aceleração progressiva dentro de cada fase
- Fruta dourada com 20% de chance de aparecer — concede um escudo que absorve o próximo golpe fatal e destrói obstáculos
- Placar em tempo real com recorde salvo no navegador
- Controle de zoom (+ / −) para monitores de diferentes tamanhos
- Cobra aguarda input antes de se mover ao iniciar cada fase
